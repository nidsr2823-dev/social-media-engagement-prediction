# Social Media Engagement Prediction

Machine Learning project that predicts total social media interactions using Random Forest Regression.

## Overview

This project analyzes social media post data and predicts total interactions based on engagement metrics and content characteristics.

## Dataset Features

- Platform
- Content Type
- Category
- Views
- Saves
- Follower Count
- Engagement Rate
- Hour of Day
- Day of Week
- Hashtag Count
- Content Length
- Sentiment
- Influencer Tier
- Has Media
- Is Verified

## Model Used

Random Forest Regressor

## Results

- MAE: 607.79
- R² Score: 0.9584

## Top Important Features

1. Engagement Rate
2. Views
3. Follower Count
4. Saves
5. Platform
